filter 뷰를 작성하다보면 filter 를 사용해 데이터를 조회해야할때가 많습니다. 전에 썻던 get() 과 filter() 비교글 을 보셨으면 조금 더 이해가 잘 되실겁니다. 계속 filter 에 대해 쓰는 이유는 중요하기도 하고 저도 배워야하기 때문에.. exact exact 는 정확히 일치하는 데이터를 찾습니다. 1 Product.objects.get(id__exact = 14) Product 라는 객체에 id 가 14번인 친구를 찾으라는 거겠죠? iexact iexact 는 대소문자를 구분하지않고 정확히 일치하는 데이터를 찾습니다. 1 Product.objects.get(name__iexact = 'k9 tank') 마찬가지로 Product 라는 객체에 name 이 k9 tank 라는 친구를 찾아오라는 겁니다....
