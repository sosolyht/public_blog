엘라스틱서치란?#

엘라스틱서치는 사이트내에 전문검색 (full text search) 기능을 제공해준다. 기존에는 아파치 루씬 (https://lucene.apache.org/) 라이브러리 기반으로 작동하였지만, 2021년 2월 이후 루씬에서 분리되어 아파치의 최상위 프로젝트이다.

엘라스틱서치는 REST API 형태#

GET , POST, PUT, DELETE 4가지의 메소드로 표현한다.

예를 들어

POST index1/_doc/1 { "name" : "최희택" , "age" : 28 }

index1 이라는 인덱스를 만들고 1번 도큐먼트를 생성

GET index1

index1 을 조회

GET index1/_doc/1

index1 인덱스에서 1번 도큐먼트를 조회

이런식으로 response , request 가 이루어진다.

와 아직까진 쉽다!

응답메시지도 REST API 스럽게

기본적인 http status 와 비슷하다

코드상태
