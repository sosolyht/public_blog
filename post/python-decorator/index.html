[Python] 데코레이터(Decorator)와 클로저(Clousure) 활용

함수 앞뒤에 기능을 추가해서 손쉽게 함수를 활용할 수 있는 기법 데코레이터
January 5, 2022 · Me | Suggest Changes

데코레이터(decorator)란?#

  • 함수 앞뒤에 기능을 추가해서 손쉽게 함수를 활용할 수 있는 기법
  • 클로저 (clousure)를 활용

함수 위에 @ 붙어있는 모습이 데코레이터입니다. 특이하죠? 그래도 알아두어야 나중에 이러한 코드를 볼때 이해할수있습니다.

데코레이터 활용#

@decorator_func
def function():
  print('@붙은게 데코레이터')

ex) 일반적인 함수

def login():
  print("희택님이 로그인하셨습니다.")

login()

희택님이 로그인하셨습니다.

이게 일반적인 함수입니다.

ex) 데코레이터를 붙인 함수

# 데코레이터 함수
def decorator_func(func):
  def wrapper():
    print('나는 데코레이터')
    func()
  return wrapper

# 일반함수에 데코레이터를 붙힌 함수
@decorator_func
def login():
  print('희택님이 로그인하셨습니다.')

login()

나는 데코레이터
희택님이 로그인하셨습니다.

클로저를 이용한 데코레이터#

# 데코레이터 함수 정의
def decorator_func(func):
  def wrapper():
    print('나는 데코레이터')
    func()
  return wrapper

# 데코레이터를 줄 함수
def login():
  print('희택님이 로그인하셨습니다.')

login()

희택님이 로그인하셨습니다.

클로저 활용

deco_func = decorator_func(login)
deco_func()

나는 데코레이터
희택님이 로그인하셨습니다.
