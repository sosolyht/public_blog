기타 설정

ConsoleMe 의 Makefile 에는 pip 명령어만 인식하기때문에,

맥에서 기본으로 지정되는 pip3 -> pip 로 변경해야됩니다.

vim ~/.bashrc

alias pip=“pip3”

입력하고

저장 후 종료

source ~/.bashrc 로 초기화

python -m pip install --upgrade pip

로 pip 업그레이드
