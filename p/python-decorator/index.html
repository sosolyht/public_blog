데코레이터(decorator)란?

  • 함수 앞뒤에 기능을 추가해서 손쉽게 함수를 활용할 수 있는 기법
  • 클로저 (clousure)를 활용

함수 위에 @ 붙어있는 모습이 데코레이터입니다. 특이하죠? 그래도 알아두어야 나중에 이러한 코드를 볼때 이해할수있습니다.

# 데코레이터 함수
def decorator_func(func):
  def wrapper():
    print('나는 데코레이터')
    func()
  return wrapper

# 일반함수에 데코레이터를 붙힌 함수
@decorator_func
def login():
  print('희택님이 로그인하셨습니다.')

# 데코레이터 함수 정의
def decorator_func(func):

    print('나는 데코레이터')
    func()
  return wrapper

  print('희택님이 로그인하셨습니다.')
