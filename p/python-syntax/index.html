python 기초 문법

print

print 는 무언가를 출력할때 사용됩니다. 예를 들어,

1

print("코알못 블로그")

라고 쓰면 코알못 블로그 라고 출력이 됩니다.

이와 같이 뭔가를 출력할때 사용되고, 결과값을 확인하기 위한 용도로도 사용됩니다.    

변수 (variable)

변수(variable)란 데이터를 저장하기 위해 할당받은 하나의 공간이라 생각하시면됩니다.

변수는 자신이 원하는 역할을 부여할수있습니다

1
2
3
4

name = "앙모띠"
age = "32"
my_car = "Tico"
live = "Seoul"

name 은 이름인 ‘앙모띠’, age는 나이인 ‘32’, my_car 는 자동차인 ‘Tico’, live는 사는 곳인 ‘Seoul’ 이런식으로 원하는 변수명으로 지정할수있습니다.

다만, 변수명을 지정할때는 조심하셔야할게있습니다. 올바른 변수 이름 : name, _name, my_name, myName 잘못된 변수 이름: 7name, my name 참고하시면 됩니다.    

데이터 타입

string (문자열)

큰따움표("") 혹은 작은따움표(’’) 안에 있는 글자들은 string 입니다. 예) “안녕하세요” , “1995” , ‘반갑습니다’ , ‘1128’

integer (정수)

정수형(Integer)이란 말 그대로 정수를 뜻하는 자료형을 말합니다.

1
2
3

a = 1
a = -100
a = 0

float (실수)

실수형(Floating-point)은 소수점이 포함된 숫자를 말합니다.

1
2

a = 0.1
a = -3.56

boolean

boolean은 참(True)과 거짓(False)을 나타내는 자료형입니다.

1
2

a = True
b = False
