CSS position 요약

position 속성을 통해 문서 상에 요소를 배치하는 방법 top,right,bottom,left 속성을 통해 요소의 최종 위치를 결정* top : 요소의 position 기준에 맞는 위쪽에서의 거리(위치)를 설정* bottom : 요소의 position 기준에 맞는 아래쪽에서의 거리(위치)를 설정* left : 요소의 position 기준에 맞는 왼쪽에서의 거리(위치)를 설정* right : 요소의 position 기준에 맞는 오른쪽에서의 거리(위치)를 설정

position 속성

position: relative

position 속성을 relative 로 설정하게 되면, 요소를 원래 위치에서 벗어나게 배치할 수 있습니다. 위치 지정은 top,bottom,left,right 를 이용해 정할 수 있습니다.

아래의 예시를 보시면 position은 relative 위치는 top: 28px; , left: 48px 로 이동하여 겹쳐있는것을 볼 수 있습니다.

1
2
3
4
5

#relative {
  position: relative;
  top: 28px;
  left: 48px;
}

 

position: absolute

position은 어렵게 생각할것없이 원래의 위치와 상관없이 위치를 지정할 수 있습니다. absolute 는 절대적이다라는 뜻때문에 마음껏 바꿀수있다고 오해하시는 경우가 있습니다. 하지만 absolute 는 부모 요소의 기준으로 위치가 결정 됩니다.

아래의 예제를 보시면 top: 80px; , right: 16px; 의 값을 주게 되면 relative 기준으로 떨어져있는 것을 보실 수 있습니다.

  position: relative;

.sample {
  width: 200px;

  right: 16px;
}

 

position: fixed

fixed 는 화면을 위아래로 스크롤하더라도 브라우저 화면의 특정 부분이 고정되어 움직이지 않습니다. 쉽게 예를들어 웹사이트들어가면 귀찮게 따라다니는 사이드 광고를 생각하시면 될것 같습니다.

아래의 예시를 참고하시면 될것같습니다.

1
2
3
4
5

#fixed {
  position: fixed;
  bottom: 8px;
  right: 16px;
}

   

display 속성

inline

inline 특징으로는 <span>,<a>,<em> 태그를 예로 들 수 있습니다.

1
2
3

<a>A</a>
<span>SPAN</span>
<em>EM<em>

을 작성해보시면 띄어쓰기가 되지않고 한 줄로 보이게됩니다. 그리고 width 와 height 속성을 지정해도 무시되는걸 참고하시길 바랍니다.  

inline-block

inline-block 은 말 그대로 inline 의 특징과 block 의 특징을 모두 가진 요소를 뜻합니다.

  • 줄바꿈이 이루어지지않습니다.
  • width 와 height 를 지정 할 수 있습니다.
  • 만약 width 와 height 지정하지 않을 경우, inline 과 같이 컨텐츠만큼 영역이 잡힌다.  

block

block 특징으로는 <div>,<p>,<h1> 태그를 예로 들 수 있습니다.

1
2
3

<div>DIV</div>
<p>P</p>
<h1>H1</h1>

을 작성해보시면 태그마다 줄바꿈되어 보여지실겁니다. block 은 inline 과는 달리 width,height 등 속성이 모두 반영이 됩니다.
